# PNG Icons Needed for FTC DECODE Scorer

Place these PNG files in `src/main/resources/images/`

## Required Icons (Recommended Size: 128x128 or 256x256 PNG)

### Game Elements
1. **artifact_green.png** - Green ARTIFACT icon
2. **artifact_purple.png** - Purple ARTIFACT icon
3. **obelisk.png** - OBELISK icon (shows MOTIF)
4. **base.png** - BASE icon (where robots return)
5. **goal.png** - GOAL icon (where ARTIFACTS are scored)
6. **ramp.png** - RAMP icon (where PATTERNS are built)

### Scoring Areas
7. **classified.png** - CLASSIFIED area icon (3 pts)
8. **overflow.png** - OVERFLOW area icon (1 pt)
9. **depot.png** - DEPOT area icon (1 pt)

### MOTIF Patterns (64x64 recommended)
10. **motif_ppg.png** - Purple-Purple-Green pattern
11. **motif_pgp.png** - Purple-Green-Purple pattern
12. **motif_gpp.png** - Green-Purple-Purple pattern

### UI Icons (32x32 or 48x48)
13. **robot.png** - Robot icon
14. **timer.png** - Timer/clock icon
15. **launch_line.png** - Launch line icon (for LEAVE scoring)

### App Icons
16. **app_icon.png** - Main application icon (512x512 for high DPI)
17. **decode_logo.png** - Official DECODE logo (if available from FIRST)

## Optional Enhancement Icons

## Alternative: SVG to PNG Conversion
If you only have SVG files, you can convert them using tools like:
- ImageMagick: `convert icon.svg -resize 128x128 icon.png`
- Inkscape: Export PNG from SVG
- Online converters: cloudconvert.com, svgtopng.com
